About:

On this site you will find four pages: Browse all Campsites, Suggested Campsites, Search Sites, and Map.

The Browse all Campsites page returns a list of all all the campsites compiled for this project.

The Suggested Campsites page highlights a few locations with multiple campsites that are most popular among visitors to Washington State. The page includes a picture of the location and a quick description.

The Search Sites page allows users to search a site by location or allows a user to filter by site type or feature.

Finally, the Map page allows users to look at sites on a map of Washington state. The sites are marked with a red marker and clicking on that marker gives the user more details about the site.
